Tính năng này cho phép bạn nhập danh sách giáo viên hàng loạt từ file Excel.

Nhập danh sách giáo viên Tải mẫu Excel giáo viên
Hướng dẫn sử dụng

  1. Tải mẫu Excel học sinh từ liên kết Tải mẫu Excel học sinh.
  2. Mở file Excel và nhập thông tin học sinh theo đúng định dạng.
  3. Lưu file Excel.
  4. Quay lại trang nhập học sinh và tải lên file Excel đã điền thông tin.
  5. Hệ thống sẽ xử lý và nhập danh sách học sinh hàng loạt.

Lưu ý:

  • Các trường bắt buộc: Tên, Họ, Email, Số điện thoại.
  • Định dạng ngày sinh: YYYY-MM-DD (ví dụ: 2010-05-15).
  • Trạng thái bán trú: "không", "bán trú", hoặc "nội trú".
  • Mỗi email học sinh phải là duy nhất trong hệ thống.

  1. Tải mẫu Excel giáo viên từ liên kết Tải mẫu Excel giáo viên.
  2. Mở file Excel và nhập thông tin giáo viên theo đúng định dạng.
  3. Lưu file Excel.
  4. Quay lại trang nhập giáo viên và tải lên file Excel đã điền thông tin.
  5. Hệ thống sẽ xử lý và nhập danh sách giáo viên hàng loạt.

Lưu ý:

  • Các trường bắt buộc: Tên đăng nhập, Email, Mật khẩu, Tên, Họ.
  • Mật khẩu phải có ít nhất 6 ký tự.
  • Tên đăng nhập và email của mỗi giáo viên phải là duy nhất trong hệ thống.
  • Vai trò mặc định là "teacher" (giáo viên).
